# Joltech — Sitio corporativo (HTML/CSS)

Versión estática del sitio corporativo Joltech, construida en HTML y CSS puros.

## Contenido
- `index.html` — Home (hero, servicios, stats, CTA)
- `servicios.html` — Portafolio de servicios y proceso
- `casos.html` — Casos de éxito
- `nosotros.html` — Sobre Joltech, misión y valores
- `contacto.html` — Formulario y datos de contacto
- `styles.css` — Sistema de diseño (paleta navy + teal)
- `assets/` — Logo e imágenes

## Uso
Abre `index.html` directamente en el navegador, o sirve la carpeta con cualquier servidor estático:

```bash
python3 -m http.server 8080
# o
npx serve .
```

## Paleta
- Navy: `#0f2e5f`
- Navy profundo: `#0a2148`
- Teal accent: `#2de2c2`
- Tipografía: Sora (display) + Inter (sans)
